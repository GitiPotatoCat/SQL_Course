## Module 1 Exercises 
Try these in IDE with level up with your Hands-On.

### Exercise 1: 
Basic **SELECT** Write a query that returns only the `FirstName`, `LastName` and `HireDate` of all employess. 

### Exercise 2: 
**Filtering** Write a query that returns all employees in the `HR` department. 

### Exercise 3: 
**Multiple conditions** Write a query that finds all active employees (`IsActive = 1`) who earn more than ₹75,000. 

### Exercise 4: 
__NULL handling__ Write a query that returns all employees where the `Department` is unknown (NULL). Then write a second query that shows their department as `'Not Assigned'` instead of NULL. 

### Exercise 5: 
__Computed column__ Write a query that shows each employee's full name as a single column called `FullName` 
(Hint: use `+` to concatenate strings, and you'll need a space `' '` between them), plus their `Salary` and a computed column `YearlySalaryWithBonus` that adds 10% to their salary.
